# 快速將docx轉換為pdf (pip install docx2pdf)
# 如果 pyinstaller 封裝為EXE後無法執行(閃退) docx2pdf的問題可以下列方式進行修改
#  1. 開啟或建立 hook-docx2pdf.py，檔案位於 ~\venv\Lib\site-packages\PyInstaller\hooks\hook-docx2pdf.py
#  2. 將檔案內容改為如下: 
     from PyInstaller.utils.hooks import collect_all
     datas, binaries, hiddenimports = collect_all('docx2pdf')
#  3. 如仍有問題，至BUILD目錄，清(移)除與此專案相關的目錄
